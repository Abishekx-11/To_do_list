# To-Do List (Command Line)

## Description
A command-line to-do list application with persistent storage. Add, view, delete, and mark tasks as complete. All tasks are saved to a file and loaded when you restart the program.

## Features
- Add tasks
- View all tasks
- Delete tasks
- Mark tasks as complete
- Tasks persist between sessions

## How to Run

python To_do_list.py

## How to use
- Download or copy code 
- Save file with name [ To_do_list.py] 
- Run python To_do_list.py in your Terminal
- After that, You will be shown options that you may want to perform like add_task, delete_task, show_task and mark task that yo completed
- Use as you wish.
